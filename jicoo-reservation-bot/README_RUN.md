# 実行方法

## 簡単な実行方法

### 1. バッチファイルを使用（推奨）

プロジェクトのルートディレクトリ（`jicoo-reservation-bot`）で以下のファイルをダブルクリックするか、コマンドプロンプトから実行してください：

- **`run.bat`**: アプリケーションを起動します

  - JAR ファイルが存在しない場合は自動的にビルドを実行します

- **`build.bat`**: アプリケーションをビルドします
  - ビルドのみを実行したい場合に使用します

### 2. コマンドラインから実行

```bash
# ビルド
mvn clean package -DskipTests

# 実行
java -jar target\jicoo-reservation-bot-1.0.0.jar
```

または

```bash
# Mavenから直接実行
mvn exec:java
```

## 注意事項

- Java 17 以上がインストールされている必要があります
- 初回実行時は、依存ライブラリのダウンロードに時間がかかる場合があります
- ウィンドウを閉じると、アプリケーションはシステムトレイに最小化されます
