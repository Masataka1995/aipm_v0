# GUI アプリケーション使用方法

## 起動方法

### Maven 経由で起動

```bash
mvn exec:java
```

### JAR ファイルから起動

```bash
java -jar target/jicoo-reservation-bot-1.0.0.jar
```

## GUI 画面の説明

### 画面構成

```
┌─────────────────────────────────────┐
│  Jicoo 自動予約 BOT                  │
├─────────────────────────────────────┤
│  状態: 待機中                        │
│  [プログレスバー]                    │
├─────────────────────────────────────┤
│                                      │
│  [ログ表示エリア]                    │
│  - リアルタイムでログを表示           │
│  - スクロール可能                     │
│                                      │
├─────────────────────────────────────┤
│  [開始]  [停止]                      │
└─────────────────────────────────────┘
```

### 機能説明

#### 1. タイトルとステータス表示

- アプリケーション名と現在の状態を表示
- 状態の色分け：
  - **青**: 待機中
  - **オレンジ**: 実行中
  - **緑**: 完了
  - **赤**: エラー
  - **グレー**: 停止中

#### 2. プログレスバー

- 実行中はアニメーション表示
- 待機中・停止中は非表示

#### 3. ログ表示エリア

- リアルタイムでログを表示
- タイムスタンプ、ログレベル、メッセージを表示
- 自動スクロール（最新ログを表示）
- 黒背景・緑文字のターミナル風デザイン

#### 4. 開始ボタン

- 予約監視を開始
- 実行中は無効化（グレーアウト）

#### 5. 停止ボタン

- 予約監視を停止
- 待機中は無効化（グレーアウト）

## 使用方法

### 基本的な使い方

1. **アプリケーションを起動**

   - `mvn exec:java` または `java -jar target/jicoo-reservation-bot-1.0.0.jar`

2. **「開始」ボタンをクリック**

   - 予約監視が開始されます
   - ログエリアに処理状況が表示されます

3. **ログを確認**

   - リアルタイムで処理状況を確認できます
   - エラーが発生した場合は赤色で表示されます

4. **「停止」ボタンで停止**
   - 実行中に停止したい場合は「停止」ボタンをクリック

### 注意事項

- 実行中にウィンドウを閉じようとすると確認ダイアログが表示されます
- 予約が成功すると自動的に処理が終了します
- エラーが発生した場合は、ログを確認して原因を特定してください

## トラブルシューティング

### GUI が起動しない場合

1. Java 17 以上がインストールされているか確認
2. Maven でビルドが成功しているか確認
3. エラーメッセージを確認

### ログが表示されない場合

1. ログアペンダーの設定を確認
2. コンソールにエラーが表示されていないか確認

### ボタンが反応しない場合

1. アプリケーションがフリーズしていないか確認
2. タスクマネージャーでプロセスを確認
3. 再起動を試行
